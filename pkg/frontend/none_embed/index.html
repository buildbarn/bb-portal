No embedded frontend!

This version of bb-portal has been built with the embedded frontend disabled.

In development?

When developing bb-portal it is recomended to proxy frontend requests to the frontend development server by setting the frontendSource configuration to proxy in the config file. That way you don't have to rebuild the frontend every time you start the backend.

In production?

Please remove the build flag --//pkg/frontend:embed_frontend=False and rebuild bb-portal to embed the proper frontend.
